# PWA 部署指南

## ✅ 构建成功！

你的应用已经成功构建为PWA版本，所有文件都在 `dist` 文件夹中。

## 📁 构建文件清单

- `index.html` - 主页面
- `manifest.json` - PWA配置文件
- `icon-192.png` - 192x192图标
- `icon-512.png` - 512x512图标
- `assets/` - 应用资源文件

## 🚀 部署到服务器

### 选项1：免费部署（推荐新手）

#### Vercel（推荐）
1. 访问 https://vercel.com
2. 注册账号（免费）
3. 点击"New Project"
4. 导入你的GitHub仓库，或直接上传 `dist` 文件夹
5. 自动部署，获得HTTPS地址

#### Netlify（推荐）
1. 访问 https://netlify.com
2. 注册账号（免费）
3. 拖拽 `dist` 文件夹到网站
4. 自动部署，获得HTTPS地址

#### GitHub Pages（免费）
1. 创建GitHub仓库
2. 上传 `dist` 文件夹内容
3. 在仓库设置中启用GitHub Pages
4. 获得HTTPS地址

### 选项2：自己的服务器

如果你有自己的服务器：

1. **上传文件**：
   - 将 `dist` 文件夹中的所有文件上传到服务器
   - 确保文件结构正确

2. **配置HTTPS**（必须）：
   - PWA必须使用HTTPS
   - 可以使用Let's Encrypt免费SSL证书

3. **配置服务器**：
   - 确保服务器支持SPA（单页应用）
   - 配置正确的MIME类型

## 📱 在iOS上安装

### 安装步骤：
1. 在iPhone/iPad的Safari中打开你的网站地址
2. 点击底部的"分享"按钮（方框+向上箭头）
3. 向下滚动，找到"添加到主屏幕"
4. 点击"添加到主屏幕"
5. 确认添加

### 验证安装：
- ✅ 主屏幕出现应用图标
- ✅ 点击图标全屏打开（没有浏览器地址栏）
- ✅ 可以离线使用

## 🔧 测试PWA功能

### 在电脑上测试：
1. 使用Chrome或Edge打开网站
2. 按F12打开开发者工具
3. 查看Application → Service Workers
4. 确认PWA已注册

### 在iOS上测试：
1. 按照上述步骤安装到主屏幕
2. 断开网络连接
3. 尝试打开应用
4. 应该能正常使用（离线功能）
